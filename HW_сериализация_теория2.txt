Ответьте на вопросы:
1. Что такое json? 
2. Как в нём представляется информация?
3. Когда и зачем можно его использовать?

JSON — это сокращённое название от JavaScript Object Notation («нотация объектов JavaScript»). Это простой текстовый формат обмена данными между приложениями и системами независимо от платформы и языка программирования. JSON используется для представления структурированных данных в виде текста.
Информация в JSON представлена в виде пар ключ-значение (key-value). Ключи — это строки, значения же могут быть строками, числами, булевыми значениями (true, false), массивами или объектами. Объекты заключаются в фигурные скобки {}, а массивы — в квадратные [ ]. Например:

{
   "имя": "Иван",
   "возраст": 30,
   "адрес": {
      "город": "Москва",
      "улица": "Ленина"
   },
   "телефоны": ["+79876543210", "+79876543211"]
}
JSON широко применяется во многих областях благодаря своей простоте и универсальности:
Для передачи данных между веб-сервером и клиентом (например, при отправке AJAX-запросов).
Хранение настроек приложений и конфигураций.
Передача информации между различными сервисами и API-интерфейсами.
Легко читается человеком и легко обрабатывается компьютером.
Таким образом, JSON — это удобный способ обмена информацией, поддерживаемый практически всеми современными языками программирования и технологиями.
